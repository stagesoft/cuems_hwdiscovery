# cuems_hwdiscovery

Part of the **CUEMS** ecosystem — see the [`cuems-RELATIONS`](https://github.com/stagesoft/cuems-RELATIONS) repo for the system index, architecture diagram, and protocol/port map.

## Role

CUEMS **hardware discovery** process — inspects a box's hardware to set up the CUEMS system on it (part of the provisioning chain, alongside `cuems-iso` for the base image and `cuems-nodeconf` for cluster adoption).

Support/tooling repo; no CUEMS-specific operational gotchas recorded yet. Related provisioning context (NIC `.link` naming, GPU quirks, per-model base images) is in the cuems-common CLAUDE.md field notes.
